# CSM Onboarding Health Tracker

An AI-powered tool for Customer Success Managers.

Enter a customer's onboarding signals — usage frequency, features adopted, champion sentiment, and blockers — and get back:

- A health score out of 100
- A churn risk flag (Low / Medium / High)
- A prioritised action plan
- Suggested QBR talking points

Generated in seconds using Claude AI.
